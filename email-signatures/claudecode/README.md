# ULTRACTION Email Signatures

A collection of 4 distinctive email signature templates for ULTRACTION GENERAL CONTRACTING LLC, designed using the brand guidelines from BRAND.md.

## Templates

### 1. Industrial Minimalist (`Template1-IndustrialMinimalist.*`)

**Aesthetic:** Sharp, grid-based layout with military precision
- Black (#141414) borders and dividers
- Logo on black background (inverted)
- Monospace-style label typography
- Grid-based information layout
- Sharp corners (0px radius)

**Best for:** Project managers, engineers, technical roles

---

### 2. Executive Luxury (`Template2-ExecutiveLuxury.*`)

**Aesthetic:** Premium, high-end corporate feel
- Maroon (#2c1810) as dominant color
- Generous whitespace and elegant spacing
- Decorative corner accents
- Social media icons included
- Gradient accent bars

**Best for:** C-level executives, sales directors, business development

---

### 3. Technical Blueprint (`Template3-TechnicalBlueprint.*`)

**Aesthetic:** Architectural blueprint inspired
- Beige (#e8dcc8) background with grid pattern
- Technical annotations and measurement marks
- Dashed borders and corner accents
- Monospace labels (E-MAIL, TEL, WEB, LOC)
- Includes license number field
- Technical reference markings

**Best for:** Architects, engineers, project leads, technical roles

---

### 4. Modern Brutalist (`Template4-ModernBrutalist.*`)

**Aesthetic:** Bold, chunky, high contrast
- Black background with white text
- Maroon accent blocks
- Asymmetric split layout
- Oversized typography
- Rotated text elements
- Bold divider bars

**Best for:** Creative directors, marketing, roles that need to stand out

---

## Usage

### React Components

```tsx
import Template1 from './Template1-IndustrialMinimalist';
import Template2 from './Template2-ExecutiveLuxury';
import Template3 from './Template3-TechnicalBlueprint';
import Template4 from './Template4-ModernBrutalist';

<Template1
  name="Sarah Mitchell"
  title="Project Director"
  email="sarah.mitchell@ultraction.ae"
  phone="+971 4 123 4567"
  website="ultraction.ae"
  address="Dubai, UAE"
/>
```

### HTML Files

The `.html` files contain email-client compatible versions with inline styles. These can be:
- Copied directly into email signature settings
- Used with email marketing platforms
- Embedded in email templates
