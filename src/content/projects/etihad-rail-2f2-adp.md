# Etihad Rail 2F2 Package ADP Project

Part of the national rail network development, this project involved earthworks, drainage, and supporting infrastructure for the Etihad Rail line.

## Project Overview

The 2F2 Package ADP (Approach Duct Package) Project involved extensive earthworks, subgrade preparation, and drainage systems for a section of the Etihad Rail network in Abu Dhabi.

## Key Features

- Earthworks and subgrade preparation
- Railway drainage systems
- Approach ducts for crossings
- Fencing and security systems
- Access roads and contractor facilities

## Technical Specifications

- Earthworks volume: 2.5 million cubic meters
- Drainage: Precast concrete culverts and channels
- Subgrade: Engineered fill with compaction control
- Fence length: 45km of security fencing
