# Etihad Rail 2D Package Bridges

Bridge construction for Etihad Rail network in Fujairah.

## Project Overview

This project involved the construction of multiple bridges as part of Etihad Rail's 2D package in Fujairah. The bridges carry the rail line across various obstacles including roads, wadis, and other infrastructure.

## Key Features

- Multiple bridge structures
- Rail bridge construction
- Reinforced concrete structures
- Foundation engineering
- Integration with rail alignment
