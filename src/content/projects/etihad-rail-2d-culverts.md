# Etihad Rail 2D Package Culverts

Critical rail infrastructure project involving the construction of precast concrete culverts for the Etihad Rail network in Fujairah.

## Project Overview

This project was part of the broader Etihad Rail network development, involving the design and construction of multiple precast concrete culverts to accommodate water flow and provide crossings for the railway line.

## Key Features

- Precast concrete culvert sections
- Multiple span configurations
- Engineered for heavy rail loads
- Water flow management systems
- Earth retention structures

## Technical Specifications

- Culvert types: Box culverts and pipe culverts
- Spans: 2m to 6m clear spans
- Load capacity: Designed for UIC 71 rail loading
- Precast elements: Factory-produced quality control
- Foundation: Reinforced concrete base slabs
